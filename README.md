# Map
This repository contains details on:  React &amp; Tailwind CSS for building responsive UI.  Leaflet.js integration for interactive maps with drawing tools.  UI/UX improvements for map components and sidebar interactions.  Debugging &amp; TypeScript issues encountered during development.  Project setup best practices for scalable frontend architecture.
A React + TypeScript + Vite application that implements the Figma UI for AOI (Area of Interest) creation and displays satellite/drone imagery using a WMS map layer. Includes interactive map features, responsive UI, Tailwind CSS integration, and Playwright tests designed for core functionality validation.

Tech Stack: React, TypeScript, Vite, Tailwind CSS, Leaflet (Map), Playwright
Features: WMS imagery layer, custom UI components, map interactions, sidebar UI, tests, and a clean modular architecture.
